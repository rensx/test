一、相交路径法

在GeoGebra中有个相交路径的指令，其含义为：

相交路径（ <多边形>， <多边形> ）在两个给定多边形之间创建相交多边形（两多边形的公共区域）。

例 1.1. 如图：边长为a的两个正方形，其中一个正方形的顶点在另一个正方形的中心，并且这个正方形绕中心转动。试计算两个正方形公共区域的面积。

作图过程：

1.1. 利用正多边形工具作一个正方形 ABCDABCD ，利用中点/中心工具作出正方形的中心 EE ；以正方形的中心为圆心、边长为半径作一个圆，并在圆周上任取一点 FF ，再利用多边形指令作出第二个正方形 EFGHEFGH 。

2.2. 利用指令：

相交路径(p1,p2)

即可作出两个正方形的公共区域，对点 FF 启动动画即可。（这里的p1、p2指两个正方形)

说明：相交路径法仅适合表示两个多边形的公共区域，不适合其它图形（如圆弧、函数构成的区域）。对于两个多边形，还有一个差集指令，我们也应该了解一下，如图：

差集(t1,t2)

得到的是左侧多边形，如图所示的阴影部分：

差集(t2,t1)

得到的图形是右侧多边形，如图所示的阴影部分：

另外还有并集指令：

并集( <多边形>, <多边形> )

与代数中并集的含义基本类似。

二、不等式法

例 2.2. 将椭圆 x225+y29=1\dfrac{x^2}{25}+\dfrac{y^2}{9}=1 与椭圆 x29+y225=1\dfrac{x^2}{9}+\dfrac{y^2}{25}=1 的公共区域表示出来。

作图过程：

只须一条指令即可作出完整图形：

x² / 25 + y² / 9 ≤ 1 ∧ x² / 9 + y² / 25 ≤ 1

解释一下：在GeoGebra中，用∧、∨、¬∧、∨、¬三个符号分别表示且、或、非三个逻辑变量，注意，“∧∧”不是键盘上输入乘方的那个符号“^”，二者不同，可用辅助键盘来输入。


例 3.3. 将椭圆 x225+y29=1\dfrac{x^2}{25}+\dfrac{y^2}{9}=1 绕原点分别旋转 60°、120°60°、 120° 后公共区域围成的图形表示出来。

作图过程：

c1:x^2/25+y^/9≤1
c2:旋转(c1,pi/3)
c3:旋转(c1,2pi/3)
c1&&c2&&c3

提示：在GeoGebra中输入“∧∧”用两个连字符“&&”代替；输入"∨∨"用两个竖线“||”代替；输入“¬¬”用一个叹号“！”代替，你可以在GeoGebra中按顺序依次将上述指令复制到地址栏中观看作图过程，下同。


例 4.4. （2022年高考全国乙卷文科数学第 55 题）：

若 x，yx，y 满足约束条件 {x+y≥2x+2y≤4y≥0\begin{cases}x+y\ge 2 \\x+2y\le 4 \\y\ge 0 \end{cases} ，则 z=2x−yz=2x-y 的最大值是_________.

A.−2B.4C.8D.12A. -2\qquad\qquad B. 4\qquad\qquad C. 8 \qquad\qquad D. 12

解题过程：

1.1. 在GeoGebta的指令栏中输入：

x+y>=2&&x+2y<=4&&y>=0

确定之后，在绘图区就会出现下图：

即把可行域表示出来了。

2.2. 输入目标函数：

2x-y=m

确定之后系统提示：

创建一个关于 mm 的滑动条，并根据题目具体情况改变滑动条的范围。

特别提醒：这里的目标函数不能输入为： z=2x−yz=2x-y ，因为在GeoGebta中， x、y、zx、y、z 是有固定含义的，即作为函数的自变量使用的。

3.3. 拖动滑动条即可观察到目标函数的取值情况，显然直线 2x−y=m2x-y=m 经过 (4,0)(4,0) 时， mm 取得最大值为 88 .

三、积分法

对于函数与直线、两函数相交构成的区域，我们一般采用积分法来作出符合条件的区域。积分法我们需要熟悉GeoGebra中的两条指令：

一是积分指令：

积分( <函数> )
积分( <函数>, <变量> )
积分( <函数>, <x-起始值>, <x-终止值> )
积分( <函数>, <x-起始值>, <x-终止值>, <是否给出积分值? true|false> )

例如：计算 f(x)=sin⁡xf(x)=\sin x 在区间 [0,π][0,\pi] 上的积分。

在GeoGebra中输入下面两条指令：

f:sin(x)
积分(f,0,pi)

则绘图区中显示：

可以看出：积分指令不但可以计算出 f(x)=sin⁡xf(x)=\sin x 在区间 [0,π][0,\pi] 上的积分值（面积），同时也把 sin⁡x\sin x 与 xx 轴围成的区域表示出来了。

二是积分介于指令：

积分介于( <函数>, <函数>, <x-起始值>, <x-终止值> )
积分介于( <函数>, <函数>, <x-起始值>, <x-终止值>, <是否给出积分值? true|false> )

例如：对于 f(x)=sin⁡xf(x)=\sin x 、 g(x)=cos⁡xg(x)=\cos x ，设 A、BA、B 是其在 [0,2π][0,2\pi] 上的两个交点，计算由 f(x)、g(x)f(x)、g(x) 两函数在两交点 A、BA、B 之间围成的图形的面积。

我们在在GeoGebra中首先作出两个函数的图像，并利用交点工具作出 A、BA、B 两个点，然后输入下面一条指令：

积分介于(f,g,x(A),x(B))

即可计算出积分值为 2.832.83 ，并将区域表示出来。指令中的x(A)、x(B)指的是 A、BA、B 两点的横坐标。

提示：如果只想将区域表示出来，而不显示积分值，则只须在指令后面加个参数：false

四、多边形法

我们先看一个例子：

例 6.6. 如图：正方形的边长为 44 ，计算阴影部分的面积。

如果我们能在坐标系中把阴影部分所在的两个圆的方程写出来，显然利用不等式法即可把该阴影部分所在的区域表示出来。

下面我们介绍另外一种作法：

1.1. 利用工具作出如下图形，比较简单，不详细介绍了；

2.2. 再作出以 A、FA、F 为端点的两段圆弧，指令为：

e:圆弧(B,F,A)
k:圆弧(E,A,F)

3.3. 在代数区选中 e、ke、k 两段圆弧确定后生在一个含两段圆弧的列表l1：

l1={e,k}

4.4. 利用序列指令，在列表l1上生成若干个点（不妨设为200），指令为：

l2=序列(描点(l1,k/200),k,1,200)

5.5. 过这200个点作一个多边形，指令为：

多边形(l2)

然后对图形进行修饰，隐藏不需要显示的元素、对字母进行重命名、设置颜色属性等等。

小结：多边形法的思路是把所作区域周界所在的元素按顺序构成一个列表，在这个列表上利用序列生成若干个点，过这若干个点作多边形，即可把区域表示出来。

圆弧(A,E,J)

五、轨迹法

轨迹法的作图过程与上面多边形化的的方法类似，即把的所要表示的封闭图形的周界元素（圆弧、线段）依顺序形成列表l1，再在l1上任取一点M，再利用指令：轨迹(M+0,M)即可将封闭图形表示出来。

下面举例说明：

例 8.8. 如图，正方形的边长为 44 ，将所示的阴影部分表示出来。

作图过程：

1.1. 利用工具或指令作出如下图形：

2.2. 利用圆弧指令再作出要表示的封闭区域的周界元素：

圆弧(A,F,G)
圆弧(B,G,H)
圆弧(C,H,E)
圆弧(D,E,F)

3.3. 依次选中 p、q、r、sp、q、r、s 四段圆弧，拖到指令栏中生成列表l1;

l1={p,q,r,s}

4.4. 在列表l1上任取一点M;

M=描点(l1)

5.5. 利用轨迹指令：

轨迹(M+0,M)

将不需要的图形元素隐藏，得到如下图形：

六、综合实例

例 9.9. 已知函数 f(x)=|x|−1f(x)=|x|-1 ，曲线 x4+3x2y+y5=1x^4+3x^2y+y^5=1 ，圆 x2+(y+1)2=1x^2+(y+1)^2=1 ，试将如图所示的阴影区域表示出来。

作图过程：

1.1. 作出题目给出的函数、曲线及圆，在指令栏中直接输入函数关系式和曲线方程：

f(x)=|x|-1
eq1:x^4+3x^2 y+y^5=1
eq2:x^2+(y+1)^2=1

2.2. 作出圆心及函数与曲线的交点 A、B、C、DA、B、C、D

O=(0,-1)
A=(-1,0)
B=(1,0)
交点(f,eq2)

3.3. 对曲线 x4+3x2y+y5=1x^4+3x^2y+y^5=1 在 xx 轴上方的一段描点，方法是先对整个曲线描点，然后用条件子列取出纵坐标大于等于 00 的点列，再用折线指令产生一个列表：

l1=序列(描点(eq1, i), i, 0, 1, 0.001)
l2=条件子列(y(A) ≥ 0, A, l1)
g=折线(l2)

4.4. 作出缎 BD、CABD、CA 及圆弧 ODCODC

h=线段(B,D)
i=线段(C,A)
c=圆弧(O,D,C)

5.5. 将线段、圆弧、拆线依顺序构造一个列表l3，利用序列指令在l3上描点生成点列l4，并对点列l4进行有序化处理生成点列l5，指令为：

l3={h, c, i, g}
l4=序列(描点(l3, i), i, 0, 1, 0.01)
graph2=旅行商问题(l4)
l5=序列(描点(graph2, i), i, 0, 1, 1 / 长度(l1))

6.6. 利用多边形指令完成图形的制作。

多边形(l5)
